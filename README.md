# VintiCode

VintiCode is a web application powered by [Next.js](https://nextjs.org), designed for rapid web development and deployment. Visit the app: [https://vinticode.vercel.app](https://vinticode.vercel.app)

## Features

- Built with TypeScript and Next.js.
- Fast and live page updates (hot reloading).
- Font optimization using [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) and [Geist](https://vercel.com/font).
- Easy deployment to Vercel.

## Getting Started

To start developing locally:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

You can start editing the homepage by modifying `app/page.tsx`. Changes auto-update in your browser.
